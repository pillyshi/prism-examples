# jmed-llm 解析例

[jmed-llm](https://github.com/sociocom/jmed-llm) データセットを使った医療テキスト解析のサンプルです。
解析には [Prism](https://github.com/pillyshi/prism) を利用します。

## jmed-llm とは

jmed-llm は日本語医療テキストを対象としたデータセットです。
詳細は[公式リポジトリ](https://github.com/sociocom/jmed-llm)を参照してください。

## セットアップ

```bash
cd examples/jmed-llm
poetry install
```
